📱 Media Queries

Media queries allow us to apply CSS based on device characteristics like viewport width. Here, when max-width ≤ 768px, we change layout: columns stack, menu becomes vertical, font sizes shrink, and everything adapts.

🧩 Stack Columns Vertically

Using flex-direction: column inside media query, we transform the .columns wrapper from row to column. Each card takes full width, perfect for mobile reading.

📐 Flexible Units & Images

All images use max-width:100%; height:auto; → they never overflow. Relative units like rem, %, vw make layout fluid. No horizontal scroll on mobile!
